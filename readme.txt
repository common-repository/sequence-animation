== Description ==

Animate every element in your page by simple selecting it and then applying the animation sequence you prefer. Multiple animations like bounce, shake, move, scale, colour and more can be used one after another.

Major features :

* Animation sequence editor for multible combination of effects
* The trigger of the effect can be chosen either on window show or mouseover events
* The repeatability of the effect can be loop or once
* The duration time of each effect can also be set for each element.

== Frequently Asked Questions ==

= 1) How to Use ? =
You only need to login as administrator and then you find the sequence animation button on admin bar, click it and then choose the desired element to animate, the editor will pop up and then you can combine the desired effects for the element.
